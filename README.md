## 작업 내용
- 초기 개발환경 세팅 (절대 경로 alias 설정 / eslint, prettier 설정 / 디렉터리 구조 / api 요청 메서드를 반환하는 객체 설정)

- 각 input 컴포넌트 단위 리렌더링
- 반복되는 react hooks -> custom hooks로 분리 (useAuthInput, useButtonActivate)
